Для запуска софта нужно сделать пару простых действий 
1. Открой проект в VS Code:

File → Open Folder → выбери папку steam_tool

2. Открой терминал в VS Code:

Нажми Terminal → New Terminal

3. В терминале набери две команды:

pip install -r requirements.txt

(подожди пока установится)

python main.py

Пользуйся!
